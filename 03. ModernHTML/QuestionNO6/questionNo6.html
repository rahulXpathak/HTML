Differences Between HTML and HTML5

HTML HTML5
Uses basic structural tags like <div> and <span>. Introduces semantic elements like <header>, <footer>, <article>, and <section>.
Requires plugins like Flash for audio and video. Includes native <audio> and <video> tags for multimedia support.
No native support for graphics or drawing. Introduces the <canvas> element for rendering 2D graphics and animations.
