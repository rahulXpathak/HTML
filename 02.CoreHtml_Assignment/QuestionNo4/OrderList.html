Ordered List of HTML Tags

  1. <html>

    The root element that defines the entire HTML document. It wraps all other HTML elements on the page.

  2. <head>

    Contains meta-information about the document, such as its title, character set, and links to stylesheets and scripts. It typically includes elements like <title>, <meta>, and <link>.

  3. <title>

    Sets the title of the HTML document, which is displayed on the browser's title bar or tab. This tag must be placed inside the <head> element.

  4. <body>

    Contains the content of the HTML document that is visible to users, such as text, images, links, and other media. It is a primary container for all content elements.

  5. <h1> to <h6>

    Define headings in HTML, where <h1> represents the highest level of heading (usually the most important) and <h6> represents the lowest level. These tags help structure content hierarchically.

  6. <p>

    Represents a paragraph of text. It is a block-level element that automatically adds space before and after the content.

  7. <a>

    Defines a hyperlink, which links to another webpage or a specific part of the same page. It is commonly used with the href attribute to specify the destination URL.

  8. <img>

    Embeds an image in the document. It requires the src attribute to specify the path to the image file and the alt attribute to provide alternative text for accessibility.

  9. <ul>

    Represents an unordered list, which is typically displayed with bullet points. It contains one or more <li> (list item) elements.

  10. <ol>

    Represents an ordered list, which is typically displayed with numbered items. It also contains one or more <li> (list item) elements.

  11. <table>

    Defines a table. It contains other elements such as <tr> (table row), <td> (table data cell), and <th> (table header cell) to structure and display tabular data.

  12. <form>

    Defines an HTML form for collecting user input. It contains form elements such as <input>, <textarea>, and <button>, and can include attributes like action and method to specify how data is submitted.

  13. <input>

    Defines an input control within a form. It can be used for various types of input, such as text fields, checkboxes, and radio buttons, with the type specified using the type attribute.

  14. <button>

    Represents a clickable button, which can be used to submit forms or trigger JavaScript actions. It can contain text or HTML content.

  15. <div>

    A block-level container element used to group and style content. It does not inherently represent any semantic meaning but is useful for layout purposes.

  16. <span>

    An inline container used to group and style small sections of content, typically within a block-level element. It allows for styling and manipulation of text or inline elements.
